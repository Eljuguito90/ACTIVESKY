# 🌤️ ActiveSky - Outdoor Weather App

Aplicación web para planificar actividades outdoor con predicción climática personalizada por actividad.

**Hackathon:** 24 horas  
**Equipo:** 6 desarrolladores (3 Frontend + 3 Backend)

---

## 🎯 Features - Fase 1

- ✅ Predicción climática personalizada por actividad
- ✅ Mapa interactivo con spots de actividades
- ✅ Recomendaciones inteligentes basadas en clima
- ✅ Búsqueda de ubicaciones con Google Maps
- ✅ Cálculo de rutas a spots

---

## 🛠️ Tech Stack

### Frontend
- HTML5, CSS3, JavaScript (Vanilla)
- Google Maps API (Maps, Places, Directions)

### Backend
- Python 3.11+
- FastAPI
- Supabase (PostgreSQL)
- OpenWeather API

---
